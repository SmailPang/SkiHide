# SkiHide

**SkiHide** 是一款专为 Windows 设计的桌面窗口管理工具，  
通过快捷键或鼠标侧键，一键隐藏 / 恢复指定窗口，  
适合 **双屏用户、游戏玩家、演示场景、摸鱼（划掉）高效办公** 等使用场景。

---

## ✨ 功能特性

- 🪟 **一键隐藏 / 恢复窗口**
  - 支持指定窗口隐藏
  - 再次触发即可恢复

- ⌨️ **全局快捷键**
  - 自定义组合键
  - 系统级监听，不影响当前应用

- 🖱️ **鼠标侧键支持**
  - 支持 X1 / X2 侧键触发
  - 无需键盘操作

- 🔇 **隐藏窗口时自动静音（可选）**
  - 隐藏时关闭系统声音
  - 恢复时自动还原原状态

- 🧠 **智能窗口过滤**
  - 自动过滤系统窗口 / 无效窗口
  - 仅显示可操作的正常应用窗口

- 📦 **免安装**
  - 单文件 EXE
  - 无需运行环境

- 🔐 **隐私合规**
  - 首次启动强制提示隐私政策与免责说明
  - 不同意将无法使用软件

---

## 🖥️ 使用场景示例

- 游戏中一键隐藏攻略窗口  
- 演示 / 投屏时快速隐藏私人内容  
- 双屏办公临时收起副屏窗口  
- 录屏 / 直播前快速清理界面  

---

## 🚀 使用方法

1. 下载最新版本的 `SkiHide.exe`
2. 双击运行（首次启动需同意隐私政策）
3. 在窗口列表中选择要隐藏的窗口
4. 设置快捷键或启用鼠标侧键
5. 点击「开始监听」

---

## ⚙️ 设置说明

- **设置快捷键**
  - 点击输入框后直接按组合键即可

- **使用鼠标侧键**
  - 勾选后可用鼠标 X1 / X2 键触发

- **隐藏后关闭声音**
  - 在「设置」中开启

## 🛠️ 构建说明（开发者）

本项目使用 **GitHub Actions + PyInstaller** 自动构建 Windows 可执行文件。

### 构建环境
- Windows Server (GitHub Actions)
- Python 3.12
- PyInstaller (spec 模式)

### 自动构建规则
- 推送 `v*` tag 时自动构建并发布 Release
- 网页端 Publish Release 同样会触发构建
- 构建产物文件名自动携带版本号，例如：
  - `SkiHide-v1.3.6.exe`
  - `SkiHide-v1.3.6-windows.zip`
